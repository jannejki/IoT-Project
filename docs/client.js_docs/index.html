ABB Ventilation Controller client.js JSDoc file

This is a document file for client.js-file, which is a part of Internet of Things TX00CI63-3006 -course's group project. Group members who participated in this project:

  • Janne Kinnunen
  • Chloe Maillot

summary

Client.js is created to work as a clientside script for Ventilation Controller UI webpage, so user can control ABB Ventilation system, which is a project for Embedded Systems Programming TX00CI61-3010 course.

This file makes the UI page dynamic, it changes modes between automode and manualmode and synchronizes slider inputs and numeric inputs for a more user-friendly experience.

Client.js is meant to work with server.js, which is a serverside javascript file. You can find JSDoc files for server.js here.
